📌 給 Lark 管理員：如何設定自動帶入資料

在多維表格「支出明細」→ 自動化 → 當表單新增記錄時，發送以下格式的連結給申請人：

付款申請單.html?applicant=Lynn&dept=標案部&payee=XX有限公司&vendor=XX有限公司&reason=購買獎杯&total=15000&nature=差旅&pay=匯款

所有可用參數：

參數名稱說明範例
applicant申請人姓名Lynn
dept申請部門標案部
payee支付對象王小明
vendor廠商名稱XX有限公司
serial編號2026-001
reason事由（支出細項）購買獎杯
remark備註含運費
total合計金額（純數字）15000
nature支付性質（逗號分隔）差旅,採購
pay支付方式匯款 或 支票 或 現金

✅ 申請人收到連結 → 點開已自動填好 → 確認 → 列印 → 手簽 → 送會計
